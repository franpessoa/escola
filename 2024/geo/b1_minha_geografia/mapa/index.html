Mapa Afetivo - Francisco Pessoa - 1°EM A / 2024

Navegação - Leia com atenção

  • Para acessar o mapa, use o botão no final desta página
  • O mapa começa com um zoom muito grande. Devem ser visíveis dois marcadores: um na cidade de São José do Rio Pardo e alguns outros na cidade de São Paulo. Para analisá-los melhor role o botão do meio do mouse e use o botão esquerdo do mouse para se deslocar no mapa
  • Caso nenhum esteja visível, utilize as instruções do item acima para diminuir o zoom até que seja possível enxergar as duas cidades
  • Para obter informações sobre os marcadores clique neles usando o botão esquerdo do mouse
